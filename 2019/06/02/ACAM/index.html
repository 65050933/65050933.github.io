一类ac自动机的计数问题，先把安全点的邻接矩阵求出来，然后跑快速幂, 注意多组样例的初始化。

这里一个小的trick就是若矩阵为A，要求A^1+A^2……+A^n则构造新矩阵B11 = A， B12 = 1，B22 = 1,B12 = 0，然后跑快速幂，最后第一行的和为答案
